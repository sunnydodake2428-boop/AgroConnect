# 🌱 AgroConnect — Direct Farm to Table Platform

> Eliminating middlemen between Indian farmers and buyers. Zero commission. AI-powered.

## 🚀 Features
- 🤖 AI Crop Detection from photo (no typing needed)
- 📊 ML Price Prediction with season adjustment
- 🌦 Live GPS weather with farming advisories
- 📈 Best Time to Sell — 12-month price trend charts
- 🎤 Voice Search in Marathi & Hindi
- 🌸 Flower & Spice marketplace
- 🛒 Full order lifecycle management
- 👨‍🌾 Farmer / Buyer / Admin roles

## 💡 The Problem
600 million Indian farmers lose 70% of earnings to middlemen.  
The same produce sells in cities for 3-4× the farm price.  
AgroConnect eliminates this gap completely.

## 🛠 Tech Stack
- **Backend:** Python Flask, SQLAlchemy, SQLite
- **ML:** NumPy, scikit-learn (price prediction)
- **Frontend:** Bootstrap 5, Chart.js, Playfair Display
- **APIs:** OpenWeatherMap, Open-Meteo, Web Speech API

## ⚙️ Setup
```bash
pip install -r requirements.txt
python init_db.py
python app.py
```
